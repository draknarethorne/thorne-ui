# v0.7.0 Changes Summary

**Release Date:** February 22, 2026  
**Previous Release:** v0.6.5  
**Branch:** feature/stat-icons-v0.7.0

---

## 🎯 Highlights

### 1) `thorne_drak` Production Output Focus

- Updated root `thorne_drak` texture assets and naming consistency for:
  - Buttons (`buttons_*_thorne01.tga`)
  - Gauges (`gauge_inlay*_thorne01.tga` families)
  - Spell/stat icons (`spell_icons_*`, `stat_icons_*`)
- Refined shipped XML consumers for these assets in key windows, including cast and animation-related files.

### 2) Options Architecture Modernization

- Expanded and normalized `Options/` structure across Gauges, Icons, Cast, Buttons, and Slots.
- Improved variant organization and discoverability for maintainers and players.
- Added/updated option documentation to match current workflows.

### 3) Slot Pipeline Foundation (Long-Term Maintainability)

- Introduced class/theme-capable slot workflow under `Options/Slots/.Master`.
- Added maintainable structure segments for `.Classes`, `.Items`, and `.Themes`.
- Added generated class/theme slot outputs and metadata to support repeatable future tuning.

### 4) Tooling and Automation Improvements

- Updated regen scripts and wrappers for gauges, gems, icons, and slots.
- Improved script consistency, configuration handling, and regeneration workflow usability.
- Added supporting analysis and implementation documentation for ongoing maintenance.
