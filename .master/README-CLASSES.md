# Classes Guide (Slots)

This guide explains how to create class-specific slot atlases and how to pick dragitem sources for a slot.

## What a Class Is

A **class** here means a class-specific item atlas generated by `regen_thorne.py`.
It allows you to override **which item icon** is used for each slot, while keeping the same slot layout.

Class configs live in:

```
.master/classes/<Class>/.regen_thorne.json
```

These configs **patch** the master atlas definition from:

```
.master/.regen_thorne.json
```

## Quick Start: Add a New Class

1. **Create a class folder**:

```
.master/classes/Necro/
```

1. **Create `.regen_thorne.json`** with only overrides.

Example (override only chest and primary):

```json
{
  "class_name": "Necro",
  "items": [
    {
      "name": "chest",
      "src_row": 2,
      "src_col": 4,
      "source_file": "dragitem7.tga",
      "tone": { "contrast": 0.85, "brightness": 1.25 }
    },
    {
      "name": "primary",
      "src_row": 5,
      "src_col": 3,
      "source_file": "dragitem9.tga"
    }
  ]
}
```

> Only include items you want to override. Everything else stays as defined in `.master/.regen_thorne.json`.

1. **Generate the class atlas and slots**:

- `.bin/regen_thorne.bat --class Necro`

This regenerates the class atlas and the class’s theme outputs in one step.

## Finding the Right Dragitem

### Where dragitems live

Dragitem sources are stored in:

```
.master/items/
```

Each `dragitem*.tga` is a grid of 40×40 tiles (6×6 tiles). Coordinates are **1-based**.

### How to choose an item

1. **Open a dragitem file** (any image editor or viewer).
2. **Count rows/cols** (1–6). Each cell is 40×40 pixels.
3. **Pick the tile** that matches the slot (helm, chest, weapon, etc.).
4. **Set `src_row`, `src_col`, and `source_file`** in your class override.

### Example

If the icon you want is in row 3, column 2 of `dragitem4.tga`:

```json
{
  "name": "head",
  "src_row": 3,
  "src_col": 2,
  "source_file": "dragitem4.tga"
}
```

## Tone Controls (Optional)

Each override can include `tone` adjustments:

```json
"tone": { "contrast": 0.95, "brightness": 1.10 }
```

- `contrast`: 1.0 = unchanged (higher = more contrast)
- `brightness`: 1.0 = unchanged (higher = brighter)

## Logos and Special Sources

- Logos are sourced from `logo_atlas_thorne01.tga`, which lives in `.master/`.
- `items/` is reserved for dragitems only.

## Workflow Notes

- `.bin/regen_thorne.bat --class <Class>` regenerates the class atlas **and** triggers slot regeneration.
- `regen_thorne.py --class <Class>` only builds the atlas; run `regen_slots.py` separately if needed.

## Reference

- Master items list: `.master/.regen_thorne.json`
- Class examples: `.master/classes/Thorne/`, `Caster/`, `Melee/`, `Hybrid/`
