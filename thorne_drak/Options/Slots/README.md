## Overview

Equipment slot variants provide class-aware inventory slot textures. Each class type
(Caster, Hybrid, Melee) generates contextual slot art highlighting the equipment
positions most relevant to that archetype. Generated by `regen_slots.py`.

## Class Variants

| Variant | Description |
|---------|-------------|
| **Caster** | Emphasizes INT gear slots (head, wrists, fingers) |
| **Hybrid** | Balanced emphasis across melee and caster slots |
| **Melee** | Emphasizes STR/DEX gear slots (weapons, chest, arms) |
| **Thorne** | Universal set — balanced for all classes *(default)* |

## Style Sub-Variants

Each class variant contains multiple visual styles:

| Style | Description |
|-------|-------------|
| **Bronze** | Warm bronze metallic finish |
| **Gold** | Gold metallic finish |
| **Patriot** | Red, white, and blue themed |
| **Silver** | Cool silver metallic finish |
| **Texture** | Textured background style |
| **Thorne** | Thorne signature dark style *(default)* |
| **Transparent** | Clear/transparent slot backgrounds |

## Files

Each style contains:
- `item_slots_thorne01.tga` — Complete equipment slot texture atlas

## Usage

```bash
sync-option.bat Slots/Caster/Gold
```

Or manually copy `item_slots_thorne01.tga` from the desired class/style combination
to `thorne_drak/` root.

## Regeneration

Slot textures are generated from source assets using the slot pipeline:

```bash
python .bin/regen_slots.py --all         # Regenerate all class/style combinations
python .bin/regen_slots.py Thorne        # Single class variant
```

See [.bin/regen_slots.md](../../../.bin/regen_slots.md) for the complete generation guide.
