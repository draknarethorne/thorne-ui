## Overview

Icon set variants replace spell, gem, and stat icon textures. Each set provides a complete replacement
for all icon files, maintaining consistent visual style across the UI. Generated by `regen_icons.py`.

## Variants

| Variant | Source | Description |
|---------|--------|-------------|
| **Classic** | EQ Default | Original EverQuest icons — faithful to the classic era |
| **Duxa** | DuxaUI | DuxaUI community icon set — clean, readable |
| **Infiniti** | Infiniti-Blue | Infiniti-Blue community icons — vibrant theme |
| **Steamworks** | TK Steamworks | Steampunk-themed icon set |
| **Thorne** | Thorne UI | Thorne curated set — optimized for readability *(default)* |
| **WoW** | Community | WoW-inspired icon style |

## Files

Each variant contains a complete icon texture set:

| File | Purpose |
|------|---------|
| `gemicons01-03.tga` | Spell gem icons (casting bar) |
| `spell_icons_thorne01-03.tga` | Spell book icons |
| `spells01-07.tga` | Spell bar icons |
| `stat_icons_thorne01.tga` | Character stat icons |

## Usage

```bash
sync-option.bat Icons/Duxa
```

Or manually copy all `.tga` files from the desired variant to `thorne_drak/` root.

## Regeneration

Icon sets are generated from source textures using the icon pipeline:

```bash
python .bin/regen_icons.py --all              # Regenerate all variants
python .bin/regen_icons.py Thorne --labels    # Single variant with labels
```

See [.bin/regen_icons.md](../../../.bin/regen_icons.md) for the complete generation guide.
