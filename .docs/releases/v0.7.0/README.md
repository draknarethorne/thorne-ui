# Thorne UI v0.7.0 Release

This directory contains release-prep documentation for `v0.7.0`.

## 📋 Release Scope

`v0.7.0` is a feature release focused on:

- Long-term `Options/` maintainability and structure improvements
- Class-aware slot generation foundations and theme pipeline expansion
- Canonical naming/placement cleanup across key texture families (buttons, gauges, icons)
- Core shipped-window consistency updates in cast and animation flows

## 🎯 Release Positioning

This release emphasizes **production-facing `thorne_drak` output quality** while also shipping foundational scripting and structure work that improves long-term maintainability.
