# VS Code Extensions (Recommended)

These extensions align with the proposed quality tooling.

## Python
- **ms-python.python** — Python support
- **ms-python.vscode-pylance** — type checking (Pyright engine)
- **charliermarsh.ruff** — Ruff lint/format integration

## Markdown
- **davidanson.vscode-markdownlint** — Markdown linting
- **yzhang.markdown-all-in-one** — Markdown editing helpers

## Spelling
- **streetsidesoftware.code-spell-checker** — CSpell integration

## YAML / JSON
- **redhat.vscode-yaml** — YAML schema/validation

## XML
- **redhat.vscode-xml** — XML validation + formatting

## General
- **editorconfig.editorconfig** — consistent formatting across editors

## Workflow (Optional)
- **GitHub.vscode-pull-request-github** — PRs and reviews in VS Code
- **donjayamanne.githistory** — Git history browsing
- **ms-vscode.powershell** — PowerShell support (Windows)
