## 📋 Overview

Options are curated UI variants organized by window category. Each option provides a complete, tested configuration ready for immediate use. No two adventurers need identical interfaces—choose the variants that match your playstyle.

**27 Categories** | **106 Variants** | **One Unified Experience**

---

## 🚀 Quick Start

### Using Options

**Method 1: Manual Copy**
1. Navigate to the desired category folder (e.g., `Options/Gauges/Bubbles/`)
2. Copy the XML file(s) to `thorne_drak/` root
3. Reload UI in-game (`/loadskin thorne_drak 1`)

**Method 2: Sync Script (Development)**
```batch
.bin/sync-option.bat [Category] [Variant]
# Example: .bin/sync-option.bat Gauges Bubbles
```

## 🔧 How Options Work

### Variant Naming Convention

| Prefix | Meaning |
|--------|---------|
| **Standard** | EQ client baseline (unmodified) |
| **Thorne** | Thorne UI default configuration |
| **Thorne [Modifier]** | Variant of Thorne default |
| **[Other Name]** | Community or specialized style |

### Sync Workflow

Options are installed by copying files to the `thorne_drak/` root directory. The sync scripts automate this:

1. `.bin\sync-option.bat` — Syncs a specific option variant
2. `.bin\sync-thorne-ui.bat` — Resets to Thorne defaults

### Cross-Window Consistency

All Thorne variants share a visual language:
- **Gauge heights**: 15px standard, 24px large
- **Text colors**: White primary, yellow highlights
- **HP fill**: Red (200,50,50)
- **Mana fill**: Blue (80,80,200)
- **Pet fill**: Purple (200,80,200)
- **Borders**: Dark theme with subtle gradients
